# AlphaVest

AlphaVest is an AI-powered personal finance dashboard that helps users track their investment portfolio, analyze stock performance, and generate simple machine-learning based predictions for stock movement.

---

## Tech Stack (Planned)

- Backend: FastAPI (Python)
- Frontend: Next.js (React) or Streamlit
- Database: PostgreSQL
- Machine Learning: Scikit-learn / XGBoost
- Market Data: Alpha Vantage API
